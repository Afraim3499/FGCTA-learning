ATR Breakout Channel – (Parent Family: Technical – Indicator‑Based, Type: Variation)
Core Logic: Use ATR to define breakout levels above/below a moving average; enter when price
breaches the channel.
How it Works: Calculate 20‑period ATR; compute mid‑line (e.g., 20‑period EMA); set upper/lower
channel at ±k × ATR; go long/short on close outside channel.
Typical Use Case: Capturing volatility expansions while filtering out noise.
Known Users/Origin: Variation of Keltner channel; widely used by algorithmic traders.
Evidence Type: Common practice.
Confidence Level: Medium.
Ichimoku Cloud Trend Strategy – (Parent Family: Technical – Indicator‑Based, Type: Core)
Core Logic: Utilise the Ichimoku Cloud to identify trend direction, support/resistance and momentum.
How it Works: Plot Tenkan‑sen, Kijun‑sen, Senkou Span A/B; go long when price is above cloud with
Tenkan‑sen above Kijun‑sen; exit when price falls below cloud.
Typical Use Case: Longer‑term swing trades; capturing major trends in Bitcoin or ETH.
Known Users/Origin: Developed by Goichi Hosoda in Japan; widely used in FX and crypto.
Evidence Type: Institutional/Academic and common practice.
Confidence Level: Medium.
Parabolic SAR Trend Following – (Parent Family: Technical – Indicator‑Based, Type: Core)
Core Logic: Use Parabolic SAR dots to determine trend direction and trailing stop; dots below price
signal long; above price signal short.
How it Works: SAR calculation uses accelerating factor; start at 0.02 and increase on each new high/
low; exit when price crosses SAR dot and dot flips sides.
Typical Use Case: Trend following with clear exit rules; useful in trending markets.
Known Users/Origin: J. Welles Wilder indicator; widely adopted.
Evidence Type: Institutional/Academic and common practice.
Confidence Level: Medium.

or reverse breakout.
Typical Use Case: Turtle trading system; capturing big trends.
Known Users/Origin: Richard Donchian and the Turtle traders popularised; widely used by system
traders.
Evidence Type: Institutional/Academic and common practice.

Turtle Trading System – (Parent Family: Technical – Breakout, Type: Variation)
Core Logic: Use two Donchian channels (20‑day for entries, 10‑day for exits) with position sizing rules;
scale into positions as price moves favourably.
How it Works: When price exceeds 20‑day high, go long; exit when price falls below 10‑day low; use
ATR‑based position sizing; risk 2% per trade.
Typical Use Case: Capturing long‑term trends in crypto futures; replicating 1980s Turtle system in
digital assets.
Known Users/Origin: Created by Richard Dennis and William Eckhardt; some crypto funds replicate.
Evidence Type: Institutional (Turtle story) and common practice.
Confidence Level: Medium.

crossing zero indicate momentum shifts.

cross from positive to negative, go short; confirm with trend filter.
Typical Use Case: Identifying early momentum surges in altcoins.
Known Users/Origin: Technical traders; indicator from early technical analysis.
Evidence Type: Common practice.
Confidence Level: Medium.

Pivot Point Reversion – (Parent Family: Technical – Mean Reversion, Type: Variation)
Core Logic: Use daily/weekly pivot points to identify intraday support/resistance; fade moves near
these levels expecting reversion to pivot.

Typical Use Case: Day trading on liquid perpetual futures pairs.

Evidence Type: Common practice.
Confidence Level: Medium.
