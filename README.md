The performance-based trading academy. Learn, Trade, and Certify across Forex, Crypto, and Gold.

## 🔗 Live Demo
**Website:** [https://fgcta-learning.vercel.app/](https://fgcta-learning.vercel.app/)

## 🚀 Overview
The FGC Academy is an institutional-grade educational platform designed to transition traders from retail noise to professional execution.

- **42 Modules** across 10 Tiers of mastery.
- **220+ Strategies** covering Global Macro, On-Chain Analytics, and Real Yields.
- **Interactive Simulation Engines** (MSS, Liquidity, Structure).
- **Publicly Verifiable Certifications** etched into a secure ledger.

## 🛠 Tech Stack
- **Framework:** Next.js 15 (Turbopack)
- **Database:** Supabase (PostgreSQL) + Prisma ORM
- **Styling:** Vanilla CSS + Tailwind (Selective)
- **State:** Zustand + React Query
- **Deployment:** Vercel
